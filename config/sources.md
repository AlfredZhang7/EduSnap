# 新闻来源网站列表

在此文件中以无序列表形式提供需要抓取的留学新闻网站 URL。

## 使用说明

- 每行一个 URL，以 `- ` 开头
- 添加新站点时请同时在 `scraping_rules.yaml` 中配置 CSS 选择器
